Bull-Live-Encoder
=================
Bull Live Encoder is a tool for pushing rtmp to server,<br/>
which supports H264 + AAC and supports ingest from camer or desktop or some windows.<br/>

blog: BLE(http://wenjiegit.github.io/Bull-Live-Encoder/)<br/>
